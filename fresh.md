# 배포 안내 메모

## 보안 명령어

```bash
git rm --cached supabase-config.js
```

## 가장 쉽게, 딱 3단계만 하시면 됩니다.

1. 프로젝트 폴더에서 터미널 열고

```bash
git add .
```

2. 저장(커밋)

```bash
git commit -m "사이트 수정본 업데이트"
```

3. 업로드(push)

```bash
git push origin main
```

끝입니다.  
그러면 GitHub에 올라가고, Vercel이 자동으로 다시 배포합니다.

---

## 한 줄 요약

수정/새 파일 생겼을 때마다: `add → commit → push` 반복하면 됩니다.
