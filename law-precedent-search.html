법령·판례 통합 검색 서비스

국가법령정보와 판례 등 공공 법령정보를 활용해, 일상·행정 분야에서 필요한 법령과 판결을 한곳에서 찾아볼 수 있도록 설계 중인 정보 창구입니다.

제공 예정인 검색·요약 기능은 참고용 일반 정보에 해당하며, 개별 사안의 적용 여부나 법적 판단을 대신하지 않습니다. 권리·의무 관계가 실제로 문제될 경우에는 행정사 등 전문가 상담을 통해 확인하시기 바랍니다.

P1

통합 키워드 검색

자연어 키워드로 법령 명칭·조문·판례를 한 번에 검색합니다. 오타 교정 및 동의어 확장(예: 「전세 사기」→ 주택임대차보호법 관련)을 포함하고, 검색창은 화면 상단에 고정합니다.

  • 국가법령정보 API
  • 대법원 판례 API
P1

생활주제 카테고리

부동산·임대차, 가족·상속, 근로·임금, 외국인·비자, 소비자 분쟁, 행정처분·불복, 사업·법인, 형사·범죄 등 8개 주제를 아이콘 기반으로 빠르게 탐색합니다.

P1

법령 원문 조회

법령명 또는 조문번호로 조회하고, 연혁·시행일 선택을 지원합니다. 어려운 용어는 툴팁으로 간단히 설명합니다.

  • 국가법령정보 Open API
P1

판례 검색 및 요약

사건번호·키워드로 판례를 검색하고, 판시요지 요약을 자동 생성합니다. 대법원 종합법률정보 등 원문 링크와 하급심 판결 포함을 목표로 합니다.

  • 대법원 Open API
  • Claude AI 요약
P2

AI 생활법률 Q&A

「보증금 돌려받을 수 있나요?」와 같은 자연어 질문에 관련 법령·판례를 인용해 답변합니다. 면책 고지 및 전문 상담 유도 CTA를 필수로 표시합니다.

  • Anthropic API
P2

행정심판·처분 판례

행정심판 재결 등 전용 탭으로 안내합니다. 비자 거절·체류연장 불허가 등 행정사 사무와 맞닿은 영역을 중점적으로 다룹니다.

  • 법령 MCP / 행심 데이터
P3

즐겨찾기·공유

법령·판례 즐겨찾기(로컬 저장), 링크 공유, PDF 저장 기능을 제공합니다.

법령·판례 통합 검색 서비스는 단계적으로 기능을 확장할 예정이며, 상기 구성은 도입 우선순위에 따른 안내입니다. 웹페이지 이용만으로 법률 자문이나 대리관계가 성립하지 않으며, 구체적 사안은 방문·유선 등 개별 상담을 통해 확인해 주세요.
